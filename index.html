This is a simple markdown parser. You can copy and paste your markdown into HTML formatted content.

Sample Markdown

# Welcome to Simple Markdown Parser\n\nThis is a simple **markdown parser** built using _vanilla JS_ and **Tailwind CSS**.\n\n## Features\n\n- Live markdown preview\n- Copy plain text\n- Copy HTML content\n- Responsive UI\n- Lightweight and fast\n\n## Usage\n\n1. Type or paste markdown in the left panel.\n2. View live preview on the right.\n3. Copy what you need!\n\n## Example Code\n\n```js\nfunction hello() {\n  console.log("Hello, world!");\n}\n```\n\nMy favorite search engine is [Duck Duck Go](https://duckduckgo.com).

How to Use This

  1. Paste or type markdown on the left.
  2. See the HTML preview on the right.
  3. Click the copy button for plain text or HTML.
